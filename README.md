# 🐾 PawTube

A minimalist, privacy-focused YouTube web player built using the official `youtube-nocookie.com` embed.

PawTube focuses on simplicity, performance, and clean UI while respecting YouTube's embedding policies.

---

## ✨ Features

- 🎥 Official YouTube NoCookie embed playback
- 🎧 Background playback support
- 🌙 Minimalist material dark interface
- 📱 Fully responsive design
- ⚡ Lightweight and fast
- 🧩 No backend required
- 🐾 Crafted by PawDevs

---

## 🔒 Privacy & Compliance

PawTube uses the official YouTube embed player via:

https://www.youtube-nocookie.com/

This project:

- Does NOT download YouTube videos
- Does NOT store or redistribute content
- Does NOT bypass ads, DRM, or restrictions
- Streams media directly from YouTube servers

YouTube is a trademark of Google LLC.

PawTube is an independent project and is not affiliated with or endorsed by YouTube or Google.

Anyone deploying this project publicly is responsible for ensuring compliance with:

- YouTube Terms of Service
- YouTube API Services Terms (if APIs are integrated)
- Local copyright laws
